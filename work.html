We Are The Diggers

(Started on May 26th, 2014)



A couple of years ago, a collection of nearly a thousand pictures of Australian soldiers from WW1 was found in a cellar in France. This discovery got nationwide news coverage in Australia. I am working with a film-maker and fellow bootcamp student on We Are The Diggers, a website which will use collective effort to crowdsource the identification of these lost Diggers (the nickname given to Autralian soldiers).

Among other things, I'm responsible for:

  • Designing the database schema and building the back-end
  • Managing authentication and authorizations with the Pundit and Devise gems
  • Implementing a text editor for superusers to edit biographies, stored in HTML format
  • Implementing search and filtering, as well as tags for the pictures, all with dynamic display through javascript
  • Facebook Connect (Not done yet)
  • Implementing a wiki-like system to allow everyone ot contribute to the content while maintaining a high standard of quality

You can checkout the source code on Github.

Airbnb-Like Rails Website

(May 19th - May 25th, 2014)



Showpcase is an imitation of TheStoreFront made with Ruby on Rails. It's like Airbnb but with commercial spaces instead of flats. I made this with another student from my bootcamp, Le Wagon and it was our first real Rails project. We threw it together in about week. It was just for practice, to get a glimpse of the various aspect of building a website in Rails, so it's not really polished, don't judge too hard :)

Some notable features:

  • Managing authentication and authorizations
  • File upload and hosting of pictures on Amazon S3
  • Deployment on Heroku
  • Integration of Google Maps (in the shops' show view)
  • Asynchronous calls on forms using javascript
  • ActiveRecord validation (on instances of Booking)
  • Welcome emails for new users

You can checkout the source code on Github.
